Important System Rules

Critical guidelines for system operation

Task Verification

Only the original task creator can verify task completion. This ensures accountability and prevents unauthorized approvals.

Private Tasks

All users can create private tasks (personal to-do items) that are visible only to themselves for personal organization.

Department Limits

Users can only assign tasks within their department (except HODs who have cross-department capabilities).

Task Forwarding

Tasks can be forwarded to equal or lower roles within the same department to maintain organizational hierarchy.
